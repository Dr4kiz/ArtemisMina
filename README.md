# ArtemisMina
Plugin de mina para o seu Factions

## Comandos

|Comando         |Descrição                      |Permissão                    |
|----------------|-------------------------------|-----------------------------|
|/mina | Entra no mundo de mineração. |`Configurável`                      |
|/mina sair | Sair do mundo de mineração. |`N/D`                      |
